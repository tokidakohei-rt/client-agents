## メール作成ルール

### 1. 基本フォーマット

```
宛先: [クライアントのメールアドレス]
CC: tokidakohei@reiwatravel.co.jp
件名: [件名]

[本文]

[署名]
```

### 2. 件名のルール

- 簡潔で内容が分かるものにする
- 【】で重要度やカテゴリを示すことを推奨

**例**:
- `【NEWT Chat】導入に関するご確認`
- `【ご確認】Agent調整の進捗について`
- `【日程調整】オンラインMTGのご案内`

### 3. 本文のルール

- 敬語を使用（ですます調）
- 簡潔かつ丁寧に
- 箇条書きを活用して読みやすく
- 具体的なアクション依頼を明確に

### 4. 挨拶テンプレート

**冒頭**:
```
[会社名]
[担当者名] 様

いつもお世話になっております。
令和トラベルの常田です。
```

**締め**:
```
ご不明点がございましたら、お気軽にお問い合わせください。
何卒よろしくお願いいたします。
```

## 使用時の注意

1. **必ずCCを設定**: `tokidakohei@reiwatravel.co.jp`
2. **必ず署名を付与**: 上記の署名テンプレートを使用
3. **送信前確認**: メール送信は承認必要アクション（ACTION_PERMISSIONS.md参照）
4. **パーソナライズ**: クライアント情報をPipedriveから取得して適切に反映
