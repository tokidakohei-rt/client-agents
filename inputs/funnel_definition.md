# ファネル定義 - NEWT Chat契約（FY26）

## 概要

NEWT Chat契約のセールス〜オンボーディングまでのファネル定義です。

**パイプラインID**: 32  
**パイプライン名**: NEWT Chat契約（FY26）

---

## ファネル全体像

```
[セールス Phase]
商談セット → Chat導入検討 → Chat導入内諾

[導入準備 Phase]  
→ チーム発行完了 → agent作成完了 → agent調整完了 → HPに実装待ち

[契約 Phase]
→ 無料期間契約 → 有料プラン契約
```

---

## ステージ詳細

### Phase 1: セールス

#### 商談セット（Stage ID: 301）

| 項目 | 内容 |
|------|------|
| **定義** | 商談の日程が確定した段階 |
| **確率** | 0% |
| **主要KPI** | 商談数、商談化率 |
| **次ステージ条件** | 商談実施、Chat導入の検討意向確認 |

#### Chat導入検討（Stage ID: 316）

| 項目 | 内容 |
|------|------|
| **定義** | Chat導入を具体的に検討している段階 |
| **確率** | 20% |
| **主要KPI** | 検討率（商談セット → Chat導入検討） |
| **次ステージ条件** | 導入の内諾を得る |

#### Chat導入内諾（Stage ID: 311）

| 項目 | 内容 |
|------|------|
| **定義** | Chat導入の内諾を得た段階 |
| **確率** | 90% |
| **主要KPI** | 内諾率（Chat導入検討 → Chat導入内諾） |
| **次ステージ条件** | チーム発行の準備開始 |
