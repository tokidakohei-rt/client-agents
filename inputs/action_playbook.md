# アクションプレイブック

## 概要

状況に応じた推奨アクションをまとめたプレイブックです。

---

## ステージ別アクション

### Lead（リード）

| 状況 | 推奨アクション | 優先度 |
|------|---------------|--------|
| 新規リード獲得 | 24時間以内に初回コンタクト | High |
| 7日間反応なし | リマインダーメール送信 | High |
| 14日間反応なし | 電話フォローアップ | High |
| 21日間反応なし | 最終アプローチ後、Lost検討 | Medium |

### Appointment（アポイント）

| 状況 | 推奨アクション | 優先度 |
|------|---------------|--------|
| アポ確定 | 確認メール送信、資料準備 | High |
| アポ前日 | リマインダー送信 | Medium |
| アポ実施後 | お礼メール、議事録送付 | High |
| キャンセル発生 | 即座に再調整提案 | High |

### Opportunity（商談）

| 状況 | 推奨アクション | 優先度 |
|------|---------------|--------|
| 商談後1週間 | 追加情報の提供、質問対応 | High |
| ニーズ確認完了 | 提案書作成着手 | High |
| 14日間進展なし | 状況確認の電話 | High |
| 競合検討中 | 差別化ポイントの強調、追加提案 | High |

### Proposal（提案）

| 状況 | 推奨アクション | 優先度 |
|------|---------------|--------|
| 提案書送付 | 1週間後にフォローアップ | High |
| 質問・懸念あり | 即座に回答、必要に応じて再提案 | High |
| 検討中（長期化） | 期限を設けた特別条件の提示 | Medium |
| 価格交渉発生 | 上長と相談、柔軟に対応 | High |

### Won（成約）

| 状況 | 推奨アクション | 優先度 |
|------|---------------|--------|
| 契約締結 | お礼、オンボーディング計画共有 | High |
| 契約書処理中 | 進捗確認、サポート | Medium |

## ヘルスレベル別アクション

### High（80-100点）

| 目的 | アクション | 頻度 |
|------|-----------|------|
| 関係維持 | 定期的な情報提供、業界ニュース共有 | 月1回 |
| 拡大機会 | 追加ニーズのヒアリング | 四半期1回 |
| 紹介依頼 | 紹介プログラムの案内 | 適宜 |
| 事例化 | ケーススタディ作成の打診 | 適宜 |

### Medium（50-79点）

| 目的 | アクション | 頻度 |
|------|-----------|------|
| 関係強化 | 定期的なチェックイン | 2週に1回 |
| 課題把握 | 課題ヒアリング、解決提案 | 月1回 |
| エンゲージメント向上 | 活用提案、トレーニング提供 | 適宜 |
| リスク監視 | ネガティブシグナルの注視 | 常時 |

### Low（0-49点）

| 目的 | アクション | 優先度 |
|------|-----------|--------|
| 即時介入 | 電話での直接コンタクト | 最優先 |
| 課題解決 | 具体的な課題の特定と解決策提示 | 最優先 |
| エスカレーション | 上長・CSマネージャーへの報告 | High |
| 特別対応 | 必要に応じて特別条件の検討 | High |
| 最終手段 | 解約防止の最終提案 | High |

## リスク別アクション

### 競合検討中

```
即時:
1. 競合情報の収集
2. 差別化ポイントの整理
3. 追加提案の準備

1週間以内:
1. 競合比較資料の送付
2. デモ・トライアルの再提案
3. 既存顧客の成功事例共有

必要に応じて:
1. 価格の再検討
2. 追加機能・サービスの提案
3. 上長同席のMTG設定
```

### レスポンス低下

```
7日間反応なし:
1. 別チャネルでのコンタクト（電話）
2. 件名を変えてリマインダー

14日間反応なし:
1. 直接電話
2. LinkedInなど別経路でのアプローチ
3. 別担当者へのコンタクト

30日間反応なし:
1. 最終アプローチメール
2. 上長からのコンタクト
3. Lost判定の検討
```

### ネガティブフィードバック

```
即時:
1. 謝罪と傾聴
2. 具体的な課題の特定
3. 解決策の提示期限を約束

24時間以内:
1. 解決策の提示
2. 改善計画の共有
3. フォローアップ日程の設定

1週間以内:
1. 改善状況の報告
2. 追加サポートの提供
3. 満足度の再確認
```

### チャーン兆候

```
即時:
1. 電話での直接コンタクト
2. 上長・CSマネージャーへの報告
3. 緊急MTGの設定

1週間以内:
1. 課題の根本原因分析
2. カスタマイズ対応の検討
3. 特別条件の提示

必要に応じて:
1. 契約条件の見直し
2. サービスレベルの向上
3. 解約防止の最終提案
```

## コミュニケーションベストプラクティス

### タイミング

| 時間帯 | 推奨度 | 理由 |
|--------|--------|------|
| 10:00-12:00 | ★★★ | 午前の集中時間後 |
| 14:00-17:00 | ★★★ | 午後の安定時間 |
| 9:00-10:00 | ★★ | 朝一は忙しい可能性 |
| 17:00-18:00 | ★ | 退社準備時間 |
| 12:00-13:00 | ★ | 昼休み |

### 曜日

| 曜日 | 推奨度 | 理由 |
|------|--------|------|
| 火曜〜木曜 | ★★★ | 週中は落ち着いている |
| 月曜 | ★★ | 週初めは忙しい |
| 金曜 | ★★ | 週末前は急ぎ対応 |

### チャネル選択

| チャネル | 適した状況 |
|---------|-----------|
| メール | 資料添付、記録が必要、フォーマル |
| 電話 | 緊急、複雑な説明、関係構築 |
| MTG | 重要な提案、複数ステークホルダー |
| チャット | 簡単な確認、既存関係が良好 |

## アクションテンプレート

### フォローアップ電話スクリプト

```
「お世話になっております、{COMPANY}の{NAME}です。
先日の{TOPIC}について、その後いかがでしょうか？

{状況確認の質問}

{必要に応じて提案}

次回は{DATE}頃にまたご連絡させていただいてもよろしいでしょうか？」
```

### 課題ヒアリング質問

```
1. 現在の状況はいかがですか？
2. 何かお困りのことはありますか？
3. 期待と実際のギャップはありますか？
4. 改善してほしい点はありますか？
5. 他に私たちにできることはありますか？
```
